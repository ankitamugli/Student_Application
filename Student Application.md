Student Application

A simple and user-friendly Student Management Application built using Java technologies. This project helps manage student records with features like adding, updating, deleting, and viewing student details.

🚀 Features
Add new student details
Update existing student information
Delete student records
View all students
Simple and clean UI
Database connectivity using JDBC
🛠️ Technologies Used
Java
JDBC
MySQL
Servlet & JSP
HTML
CSS
Apache Tomcat
Eclipse IDE

⚙️ Setup Instructions
Clone the repository
git clone https://github.com/ankitamugli/Student_Application.git
Import the project into Eclipse
Configure Apache Tomcat Server
Create MySQL database
CREATE DATABASE student_application;
Update database username and password in JDBC connection file
Run the project on Tomcat server
💻 Database Example
CREATE TABLE student (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    email VARCHAR(100),
    course VARCHAR(100)
);
